## Using JavaScript libraries
```javascript
// either by content delivery networks (CDN)
<script
    src="https://cdnjs.cloudflare.com/ajax/libs/d3/7.1.1/d3.min.js"

// or by installing libraries locally
// my_project/
// |--static
//    |----css

//         |----d3.mi.js

<script src="/static/libs/d3.min.js"></script>
```
